# 🗑️ Waste Management System

A full-stack web application designed to improve waste reporting, tracking, and management within urban environments.

## 🚀 Overview

The Waste Management System allows users to report waste issues, track collection status, and enables administrators to manage and verify waste reports efficiently.

This project demonstrates full-stack development, authentication systems, and database integration.

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript
- React.js

**Backend:**
- Node.js
- Express.js

**Database:**
- MySQL

**Other Tools:**
- Git & GitHub
- Google Authentication
- Email Verification System

---

## ✨ Features

- 🔐 User Authentication (Google Sign-In + Email Verification)
- 📝 Waste Reporting System
- 📊 Waste Tracking Dashboard
- 🗄️ Database Integration for Data Storage
- 🔄 Real-time Frontend-Backend Communication
- 👨‍💼 Admin Verification System

---

## 🧠 How It Works

1. Users sign up/login using Google Authentication or email verification.
2. Users submit waste reports with relevant details.
3. The system stores the report in the database.
4. Admin reviews and verifies the waste reports.
5. Users can track the status of their reports.
